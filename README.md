# Suplify Fitness Website - Frontend

Next.js frontend for **Suplify Fitness**, a comprehensive platform that enables users to:
- Book appointments with doctors
- Access personalized workout plans and live workout classes
- Purchase supplements directly from the integrated online store

## Features

- Built with Next.js for server-side rendering and optimized performance  
- Responsive and mobile-friendly UI  
- Seamless navigation between appointment booking, workout content, and supplement store  
- Integration-ready for backend APIs (authentication, appointments, products, etc.)  
- Support for user profiles and subscription management
